    # 浏览器

    1. hello world 会变成红色吗？？？
    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
    <script>
      document.getElementById("app").style.color = "red";
    </script>
  </head>
  <body>
    <div id="app">
      hello world!
    </div>
  </body>
</html>

    不会，浏览器在执行渲染时，执行到script标签后，先进行js执行，执行完成后，渲染div标签；js执行时，页面中获取不到app标签，所以没有改变颜色 渲染引擎与js引擎线程互斥

    1. visibility: hidden 会发生重排或重绘吗？？？

    会发生重绘，因为只是显示 隐藏元素，dom元素并没有删除

    1. display: none 会发生重排或重绘吗？？？

    会发生重排，因为删除了元素的dom，浏览器需要重新计算并绘制页面
